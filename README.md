# AIチューターカガワさん (v1.1.0)

高校生向け、Gemini 3.1 Pro APIを利用したマルチモーダル・プロンプトAI家庭教師アプリです。
Next.js (App Router) を使用して構築されています。

## ⚠️ 【重要】GitHubへアップロードしてはいけないファイル

このアプリのソースコードをGitHubや一般のクラウド上にアップロードする際、**以下のファイルやフォルダは絶対に含めない（アップロードしない）** ようご注意ください。

1. **`.env.local`**
   - あなたの `GEMINI_API_KEY`（パスワードと同等の情報）が書かれています。これが公開されると第三者に勝手にAPIを使われ、利用枠や課金が発生する深刻なトラブルに繋がります。
2. **`node_modules/` フォルダ**
   - 膨大なライブラリが詰まったフォルダです。容量が大きすぎることと、環境によって中身が違うため、アップロードはしません（Gitでは通常無視されます）。
3. **`.next/` フォルダ**
   - アプリをビルド（動作準備）する際に自動生成される一時フォルダで、同様にアップロード不要です。

※手動でファイルをアップロードする際は、上記3つを選択から除外してください。

---

## 🚀 デプロイ（一般公開）の手順

Vercelを使用すると、非常に簡単にインターネット上へ公開できます。

1. GitHubにプロジェクトをアップロードします（上記「禁止ファイル」を除外してください）。
2. [Vercel](https://vercel.com/) にGitHubでログインし、「Add New Project」からリポジトリを選択します。
3. デプロイ設定の **「Environment Variables」** に以下を追加します。
   - Key: `GEMINI_API_KEY`
   - Value: `（あなたのGemini APIキー）`
4. 「Deploy」ボタンを押して完了です。

---

## 💻 ローカルでの開発・起動方法

1. `npm install` または `npm i` で必要なパッケージをダウンロード
2. `.env.example` をコピーして `.env.local` という名前に変更し、中に `GEMINI_API_KEY=あなたのキー` を書く
3. 以下のコマンドで起動します:

```bash
npm run dev
# または yarn dev / pnpm dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開くとアプリが表示されます。
